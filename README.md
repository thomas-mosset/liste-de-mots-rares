## **Fonctionnalités**

- Création, édition, suppression d’un mot ;
  
- Affichage des mots :
  - sous format de liste ;
  - sous format individuel ;

- Mot du mot (le mot change toutes les 24h) ;

- Rechercher un mot ;
  
- Tri des mots par :
  - date d'ajout (plus récent) ;
  - alphabétique ;
  - alphabétique inversé ;
